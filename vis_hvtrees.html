Принцип построения деревьев

  • Полные бинарные деревья
    • $c_0 = e$, $c_i = c_{i-1} ⊕ c_{i-1}$ for $i ≥ 1$
  • Деревья Фибоначчи
    • $F_0 = e$,   $F_1 = e ⊕ e$,
      $F_i = F_{i-1} ⊕ F_{i-2}$ for $i > 1$
    • нумерация вершин: $n_F(0) = 0$,  $n_F(1) = 1$,
      $n_F(i) = n_F(i-1)+n_F(i-2)+1$ for $i > 1$

Требования к вертикальной укладке

  • Вершины имеют целочисл. координаты

  • Рёбра – прямые отрезки

  • Потомок имеет большую ординату, чем родитель (ось OY направлена вниз)

Алгоритм потроения h-v-укладки бинарного дерева


def BT():
	if "$t = e$":
		"$\delta_t = \varepsilon$"
	if "$t = t_1 \oplus t_2$" and "$w_{t_1} \ge w_{t_2}$"
		"$\delta_t = \delta_{t_1} ⊘ \delta_{t_2}$"
	if "$t = t_1 \oplus t_2$" and "$w_{t_1} < w_{t_2}$":
		"$\delta_t = \delta_{t_2} ⊘ \delta_{t_1}$"
  Теорема 2. Для дерева с $n$ вершинами данный алгоритм строит h-v-укладку площади не более $(n \log n + 1) n$ за линейное время и линейную память.

Доказательство

Докажем по методу мат. индукции:
$h_{\delta_t} \le n$ $w_{\delta_t} \le \log n + 1$
  • База. Для $n = 1$ очевидно $h_1 = 1$, $w_1 \le \log 1 + 1 = 1$.
  • Гипотеза.Утверждение справедливо для деревьев на $\lt n$ вершинах.

Доказательство

  • Доказательство. Предположим:
    • $t = t_1 \oplus t_2$ - дерево с $n$ вершинами
    • пусть $w_{\delta_{t_1}} \le w_{\delta_{t_2}}$
    • деревья объединяются с помощью вертикального склеивания (для $\ominus$ предположение $h_{\delta_{t_1}} \le h_{\delta_{t_2}}$)
    Выделим 3 случая:
    1. $t_2 = e$
    2. $t_2 \neq e, w_{\delta_{t_1}} \le w_{\delta_{t_2}}$
    3. $t_2 \neq e, w_{\delta_{t_1}} = w_{\delta_{t_2}} = w$

Доказательство

  • Случай 1.
    $h_{\delta_t} = h_{\delta_{t_1}} + 1$
    $w_{\delta_t} = w_{\delta_{t_1}}$
  • Случай 2.
    $h_{\delta_t} = h_{\delta_{t_1}} + h_{\delta_{t_2}}$
    $w_{\delta_t} = w_{\delta_{t_1}}$

  • т.к. $n_1 + n_2 < n$, то гипотеза выполняется.

Доказательство

  • Случай 3.
    $h_{\delta_t} = h_{\delta_{t_1}} + h_{\delta_{t_2}}$
    $w_{\delta_t} = w + 1$
    • Из гипотезы следует: $w \le \log n_i + 1, i = \overline{1,2}$
      $n_i \ge 2^{w-1}$
      $n = n_1 + n_2 + 1 \ge 2^w + 1$
      $w \le \log(n - 1) \le \log n$
      $w_{\delta_t} = w + 1 \le \log n + 1$

Доказательство

  • Алгоритм использует линейное время и линейную память
